# -Solana-
这个脚本实现的是 Solana 链上多钱包资金的自动归集与回收，也就是你所说的“汇总账户的资金回收”。它的核心功能如下：

批量读取钱包：从一个 Excel 或 CSV 文件中读取大量钱包的私钥（例如交易所下发的用户充值地址、活动奖励钱包等）。

查询所有钱包余额：连接 Solana 主网 RPC 节点，异步查询每个钱包的 SOL 余额，并计算扣除转账手续费后可转出的金额。

智能筛选：只对余额超过最小归集阈值（默认 0.00001 SOL）的钱包执行转账，避免小额浪费 Gas 费。

一键归集到主钱包：将符合条件的所有钱包中的 SOL 资金，批量转账到你指定的主钱包地址，实现“零散余额 → 单一金库”的集中管理。

保留最低租金：脚本可配置每个钱包保留极少量 SOL（默认 5000 lamports ≈ 0.000005 SOL），避免账户被系统回收，但不影响资金回收效率。

生成汇总报告：转账完成后自动生成 Excel 结果文件，记录每个钱包的转出金额、交易签名、状态和时间，便于对账。

简单说：你只需要维护一张包含所有子钱包私钥的表格，运行这个脚本，它就会自动把分散在成百上千个钱包里的“碎钱”全部收拢到你的主钱包里。非常适合项目方、交易所或 DePIN 运营方用于每日资金结算、佣金回收或地址余额清收。
